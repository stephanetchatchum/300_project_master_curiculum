# Hello World CLI

A simple command-line interface application that collects user information and displays a personalized summary. This project demonstrates basic Python input/output operations and includes multilingual support.

## Features

- Interactive user input collection
- Personalized greeting and summary display
- Multilingual support (English, French, Kinyarwanda)
- Simple and educational code structure

## Files

- `main.py` - Basic version with English prompts only
- `exercise.py` - Enhanced version with multilingual support
- `README.md` - This documentation file

## Requirements

- Python 3.x

## How to Run

### Basic Version (English only)
```bash
python main.py
```

### Multilingual Version
```bash
python exercise.py
```

When prompted, choose your preferred language:
- `en` for English
- `fr` for French
- `rw` for Kinyarwanda

## Usage Example

```
Choose a language:(en, fr, rw)
en
What is your name? John
What's your favorite color? Blue
What's your Job? Developer
What's your Dream? Travel the world
------ Summary ------
Name: John
Color: Blue
Job: Developer
Dream: Travel the world
```
